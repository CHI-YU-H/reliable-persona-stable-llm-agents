# Reliable Persona-Stable LLM Agents

本專案包含一套應用於實驗室導覽任務的人格設定大型語言模型代理人架構之系統實作與實驗資料。

[English Version](README.md)

## 實驗內容

| 實驗 | 重點 | 說明 |
|---|---|---|
| 實驗三 | 導覽情境式輸出分析 | 觀察本研究提出之架構在實驗室導覽情境中的回答表現。 |
| 實驗四 | 內部知識庫消融實驗 | 比較代理人在有無內部實驗室知識庫支撐下的回答差異。 |
| 實驗五 | 任務延續機制消融實驗 | 比較代理人在有無任務延續判定機制下，對多輪對話脈絡的掌握差異。 |

完整實驗資料可參考 [`experiments`](experiments/)。
