# Reliable Persona-Stable LLM Agents

本專案包含一套應用於實驗室導覽任務的人格設定大型語言模型代理人架構之系統實作與實驗資料。

[English Version](README.md)

## 實驗內容

| 實驗 | 重點 | 說明 |
|---|---|---|
| 實驗三 | 多輪任務脈絡的延續能力 | 觀察代理人是否能在多輪互動中延續前後文任務脈絡。 |
| 實驗四 | 內部知識庫的使用程度 | 觀察代理人的回答是否能受到內部知識庫內容支撐。 |
| 實驗五 | 回答邊界的控制 | 觀察代理人在資料不足或問題超出可回答範圍時，是否能控制回答邊界並避免產生未被支撐的內容。 |

完整實驗資料可參考 [`experiments`](experiments/)。
